# TopTire AI - Vanilla HTML/JS Frontend

## Usage
Simply open `index.html` in a web browser. No build process required!

## Deploy
You can deploy this to any static hosting service:
- Railway Static Sites
- GitHub Pages
- Netlify
- Vercel
- Or any web server (Apache, Nginx, etc.)

## Features
- Pure HTML, CSS, and JavaScript (no frameworks)
- No dependencies or build process
- Fully standalone and portable
- Arabic language support (RTL)
- Responsive design for mobile and desktop
- Connection to Railway-deployed backend
- Error handling and loading states

## Configuration
To change the API URL, edit the `API_URL` constant in the `<script>` section of `index.html`.
