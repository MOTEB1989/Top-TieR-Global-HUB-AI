# AGENT PLAYBOOK – Top-TieR Global HUB AI

## الهدف  
تحديد طريقة عمل الوكيل الذكي، وحدود التشغيل، وطريقة التفاعل مع المستخدم.

## تعريف الوكيل  
الوكيل الذكي Smart Agent يقوم بـ:
- تحليل الطلب  
- تحديد المصدر المناسب  
- استدعاء النماذج  
- دمج النتائج  
- إرجاع استجابة موحّدة مدعومة بمصادر رسمية

## دورة التشغيل  
1. استقبال الطلب  
2. تحليل السياق  
3. تحديد نوع الوكيل (بحث – تحليل – استرجاع – توليد)  
4. اختيار النموذج المناسب (OpenAI أو Groq أو Anthropic)  
5. تنفيذ الفحوصات (connectivity checks)  
6. تنفيذ المهمة  
7. إرجاع النتيجة  
8. حفظ log داخلي (اختياري)

## حدود الوكيل  
- لا يكتب ملفات بدون إذن  
- لا ينفّذ كود خارجي خارج sandbox  
- لا يرسل مفاتيح حساسة  
- يخضع لسياسة allowlist في Telegram
